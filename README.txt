Description
===========
This E-commerce website is a static website and all data are static it shows only functions but only adding to cart is working,
which is used for adding items into the cart and checking out. The banner is onclick change contents type.

technologies used:
==================
HTML,CSS,Boottrap, javaScript

Details:
=========
this website contains multiple products section like mobile, speaker, and etc.

working modal:
==============

the user visit the website and can check multiple images banner by clicking on button on the banner.
and the user can add items to cart by clicking to add to cart button. and also can clear the cart by remove from cart.
and on clicking checkout button the user can checkout the products. and fill add details for delivery of product in delivery address form.
